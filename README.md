# Caml Break

# How to Run

Prerequisites: 
[OCaml is installed](https://ocaml.org/docs/install.html)

Install Packages:
```
opam install js_of_ocaml js_of_ocaml-ocamlbuild js_of_ocaml-camlp4 js_of_ocaml-lwt
```

Compile and Run:
```make``` and open ```index.html``` in your favorite browser
